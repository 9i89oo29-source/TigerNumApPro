# TigerNumApp - منصة الأرقام الافتراضية العربية

تطبيق أندرويد متكامل لشراء أرقام افتراضية لاستقبال رسائل التحقق، مع لوحة تحكم مشرف وبنية خلفية احترافية.

## الميزات
- واجهة عربية RTL بالكامل
- دعم Dark Mode
- مزودان للـ SMS (Hero-SMS و Tiger-SMS) مع آلية fallback
- أمان عالي: تشفير، SSL Pinning، JWT، حماية من abuse
- لوحة تحكم مشرف سهلة الاستخدام

## متطلبات التشغيل
- **Android**: Android Studio Hedgehog+
- **Backend**: Node.js 18+, MongoDB 7, Redis
- **Admin**: متصفح حديث
